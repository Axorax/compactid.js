<p align="center">Simple library to generate UUIDs</p>

## ⚙️ Installation

```js
npm i compactid
```

## 📘 Usage

#### → Import

```js
// commonjs
const compactid = require("compactid");

// ES6
import compactid from "compactid";
```

#### → Generate

```js
const id = compactid();

console.log(id);
```
